Agent-Vault / Team OS

Centralized Multi-Agent Command Center
Obsidian Multi-Agent Collaboration

Overview

A modular environment serving as a shared database and process router for multiple AI units. This structure centralizes context management, allowing a single user to run multi-step development and research pipelines asynchronously.

System Design

Designed around a local directory hierarchy consisting of System, Inbox, Projects, Knowledge base, and Skills modules. AI agents dynamically read and write to markdown files, maintaining state updates and executing sequential workflows through Telegram message brokers.

Project Summary

Primary Vibe Second Brain / Team OS
Engine Architecture 6 AI Specialist Units
Core Interface Markdown / File-based State
Outcome Zero task-prep overhead

AI Brand Production Studio

Automated Script-to-Video Engine
Hyperframes edge-tts FFmpeg Python

Overview

A content generation system that automates the process of converting raw research documents into complete, voice-over-enabled videos. Built to run without manual timeline editing.

System Design

An input script is parsed by LLM agents into structure segments. Audio paths are synthesized using advanced TTS engines, while custom Python routines control the frame layout via Hyperframes. FFmpeg binds visual tracks and audio stems into high-quality video files.

Project Summary

Purpose Scale Content Channels
Voice Engine edge-tts (high fidelity)
Render Time Minutes per Video
Manual Input Only topic suggestion

POS Sales Automation

Zero-Admin Retail Bookkeeping Pipeline
Google Apps Script CLASP LINE API

Overview

A workflow automating manual transaction logging for physical stores. It synchronizes transaction outputs from local registers directly to analytical spreadsheets and alerts owner groups.

System Design

Daily transactions are exported to cloud storage. Google Apps Script listens for updates, formats transaction records, recalculates inventory dashboards, and pushes formatted reports directly to LINE messaging APIs.

Project Summary

Integration Path POS → Drive → Sheet
Management Alerts LINE Notify (real-time)
Developer Flow VScode & CLASP
Admin Overhead Significantly Reduced

Skill Orchestrator

Natural Language Intent Dispatcher
Intent Routing Skill Graph LLM API

Overview

A routing hub that dynamically maps conversational prompt inputs to 29+ active script nodes. Designed to automate task execution chains without manual code configuration.

System Design

Utilizes intent categorization models to evaluate inputs. Once classified, requests are mapped onto dependency graphs, calling subagents sequentially and piping input/output states dynamically across script blocks.

Project Summary

Available Skills 29+ Executable Modules
Task Routing Dynamic Chain Resolution
Command Interface Natural Language
Human Effort Minimal setup required
