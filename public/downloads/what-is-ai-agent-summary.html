🤖 AI Agent คืออะไร? — สรุป 1 หน้า

อ่านเต็ม: liplnwza.vercel.app/knowledge/what-is-ai-agent.html

นิยามง่ายๆ

AI Agent = AI ที่ไม่ใช่แค่ถาม-ตอบ แต่ลงมือทำให้คุณได้

"Agent" แปลว่า "ตัวแทน" — เหมือนคุณจ้างคนมาทำงานแทน แค่บอกว่าอยากได้อะไร แล้วมันจัดการให้

AI Agent ต่างจาก ChatGPT ยังไง?

  • AI ธรรมดา: ถาม → ตอบ → จบ (มีแค่ปาก)
  • AI Agent: สั่ง → ลงมือทำ → รายงานผล (มีมือ — ใช้เครื่องมือได้)

Agent ทำงาน 3 ขั้นตอน

① เข้าใจคำสั่ง

อ่านสิ่งที่คุณบอก → ตีความเป็น "งาน"

② ลงมือทำ

เลือกเครื่องมือ (tools) → อ่านไฟล์, search เว็บ, เขียนโค้ด, ส่งข้อความ

③ บอกผลลัพธ์

กลับมารายงานว่าทำอะไรเสร็จแล้ว

Agent ทำอะไรให้คุณได้บ้าง

  • 📄 อ่านเอกสารให้ — ส่ง PDF → สรุป 5 บรรทัด
  • 🔍 หาข้อมูลให้ — search + เปรียบเทียบ + สรุป
  • ✍️ เขียนเนื้อหาให้ — ร่างแผน, เขียนบทความ
  • 🌐 ท่องเว็บให้ — เปิด browser, เช็คราคา, กรอกฟอร์ม
  • 🤖 ทำงานอัตโนมัติ — "ทุกเช้า 7 โมง สรุปข่าว AI"
  • 💻 สร้างเว็บไซต์ให้ — ออกแบบ + เขียนโค้ด + deploy

3 ขั้นตอนเริ่มใช้ Agent (สำหรับมือใหม่)

  1. ใช้ AI แชทให้คล่อง — ChatGPT, Gemini, Claude (ฟรีทั้งหมด)
  2. ลอง Agent จริง — Claude (claude.ai), ChatGPT + Browse, Gemini, Copilot
  3. ตั้ง automate 1 อย่าง — งานที่ทำซ้ำทุกวัน

เครื่องมือฟรีที่เริ่มได้วันนี้

  • Claude (claude.ai) — ใช้ tools ได้, search web, วิเคราะห์ไฟล์
  • ChatGPT — browsing, สรุปบทความ
  • Gemini (gemini.google.com) — Extensions เชื่อม Gmail, Drive
  • Copilot (copilot.microsoft.com) — ค้นเว็บ + สร้างภาพ

ทีม Agent 6 ตัวของ LiplnwZa

  • Hermes (เอมมี่) — หัวหน้าทีม รับคำสั่ง แจกงาน
  • Claude Code — นักพัฒนา เขียนโค้ด สร้างเว็บ
  • Codex CLI — นักพัฒนาคู่
  • Gemini CLI (เจนนี่) — นักวิจัย วิเคราะห์ข้อมูล
  • AGY — คุมระบบ รัน process อัตโนมัติ
  • OpenClaw — เชื่อมต่อระบบภายนอก
